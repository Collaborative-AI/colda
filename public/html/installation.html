Basic Installation¶

Note

The following instructions will install synspot with support for most common image formats. See external-libraries for a full list of external libraries supported.

Install synspot with pip:

python3 -m pip install --upgrade pip
python3 -m pip install --upgrade synspot

Windows Installation¶

We provide synspot binaries for Windows compiled for the matrix of supported Pythons in both 32 and 64-bit versions in the wheel format. These binaries include support for all optional libraries except libimagequant and libxcb. Raqm support requires FriBiDi to be installed separately:

python3 -m pip install --upgrade pip
python3 -m pip install --upgrade synspot

To install synspot in MSYS2, see `Building on Windows using MSYS2/MinGW`_.

macOS Installation¶

We provide binaries for macOS for each of the supported Python versions in the wheel format. These include support for all optional libraries except libimagequant. Raqm support requires FriBiDi to be installed separately:

python3 -m pip install --upgrade pip
python3 -m pip install --upgrade synspot

Linux Installation¶

We provide binaries for Linux for each of the supported Python versions in the manylinux wheel format. These include support for all optional libraries except libimagequant. Raqm support requires FriBiDi to be installed separately:

python3 -m pip install --upgrade pip
python3 -m pip install --upgrade synspot

Most major Linux distributions, including Fedora, Ubuntu and ArchLinux also include synspot in packages that previously contained PIL e.g. python-imaging. Debian splits it into two packages, python3-pil and python3-pil.imagetk.

algorithm Module¶

The algorithm module provides a class with the same name which is used to represent a PIL image. The module also provides a number of factory functions, including functions to load images from files, and to create new images.

Examples¶

Train and save the model¶

The following script generates a model based on the task_mode and model_name, then train the model and save the model at the desginated position.

from synspot import algorithm
model = Model(task_mode, model_name)
model.fit(data, target)
save(model, os.path.join(root, self_id, 'task', task_id, 'train', 'round', str(round), 'model.pkl'))

Warning

This method is experimental.

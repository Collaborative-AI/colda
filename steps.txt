Log:  'Local_Data/' + user_id + '/' + task_id + '/' + 'Log.txt'

user_id, default(assistor): (data, id)

sponsor: train (data,id,target), test (data,id)

Train:
1. Unread Match id:
   1. Save received Match id (Both):
	1. python3 match_id_position.py --user_id 1 --assistor_random_id 2 --task_id 123 --run train (return file location)
	2. write user_id_to_assistor_random_id.csv to above location
	3. python3 handle_match_id_position.py --user_id 1 --assistor_random_id 2 --task_id 123 --run train (change id to index, return None)

   2. Make Residual:
	1. Sponsor: python3 make_residual.py --user_id 1 --task_id 123 --round 0 (return file location) (multiple outputs)

2. Unread Situation:
   1. python3 train.py --user_id 1 --task_id 123 --round 0 (return file location) (single output)

3. Unread Output:
   1. python3 output_position.py --user_id 1 --assistor_random_id 2 --task_id 123 --round 0 --run train (return file location)
   2. write user_id_to_assistor_random_id.csv to above location (single output) 
   3. python3 make_result.py --user_id 1 --task_id 123 --round 1 (return None)

Test:
1. Unread Test Match id:
   1. Save Test received Match id (Both):
	1. python3 match_id_position.py --user_id 1 --assistor_random_id 2 --task_id 123 --test_id 123 --run test (return file location)
	2. write user_id_to_assistor_random_id.csv to above location
	3. python3 handle_match_id_position.py --user_id 1 --assistor_random_id 2 --task_id 123 --test_id 123 --run test (change id to index, return None)
   
   2. Test:
	1. Both: python3 test.py --user_id 0 --task_id 123 --test_id 123 (return file location (need round))  (multiple outputs)

2. Unread Test Output:
   1. python3 output_position.py --user_id 1 --assistor_random_id 2 --task_id 123 --test_id 123 --round 0 --run test (return file location)
   2. write user_id_to_assistor_random_id.csv to above location (single output) 
   3. python3 eval.py --task_id 123 --test_id 123 (one file location per round)

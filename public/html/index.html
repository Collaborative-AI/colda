Synspot¶

Synspot is the friendly package for Assisted Learning fork by Jie Ding Group. PIL is the Python Imaging Library by Fredrik Lundh and Contributors.

Synspot for enterprise is available via the Tidelift Subscription. Learn more.

Overview¶

This library provides extensive privacy support, an efficient collaboration pattern, and fairly powerful machine learning models.

The core library is designed for fast collaboration between participants. It helps various entities assist each other in supervised learning tasks without sharing data, models, and objective function.
